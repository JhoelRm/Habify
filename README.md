# Habify

## Descripción

Habify es una aplicación móvil orientada a la mejora de hábitos diarios, con énfasis en la alimentación y el bienestar general. Permite al usuario registrar información relevante, dar seguimiento a sus actividades y observar su progreso de manera clara, con el objetivo de facilitar cambios sostenibles en el tiempo.

## Propósito

El propósito del proyecto es brindar una herramienta práctica que ayude a las personas a desarrollar hábitos más saludables mediante el registro, el seguimiento continuo y la retroalimentación, considerando tanto la salud física como el bienestar mental.

## Funcionalidades

* Registro del perfil del usuario
* Registro diario de hábitos y alimentación
* Control básico de consumo (calorías, agua, actividad)
* Recordatorios para mantener la constancia
* Visualización del progreso
* Recomendaciones generales adaptadas al usuario

## Requerimientos

### Funcionales
